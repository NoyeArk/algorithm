# 309. 买卖股票的最佳时机含冷冻期

> **日期**：2024-02-06  
> **所用时间**：53min  
> **知识点**：动态规划、状态机

## 1. 题目描述

给定一个整数数组 `prices`，其中 `prices[i]` 表示第 `i` 天的股票价格。设计算法计算**最大利润**。

**约束条件：**

- 不能同时参与多笔交易：再次买入前必须先卖出之前的股票。
- **冷冻期**：卖出股票后，**第二天不能买入**（即至少休息一天才能再买）。

**示例：**

- **输入**：prices = [1,2,3,0,2]
- **输出**：3
- **解释**：对应交易序列为：第 1 天买入、第 2 天卖出、第 3 天冷冻、第 4 天买入、第 5 天卖出。利润 (2-1) + (2-0) = 3。

**约束：**

- $1 \le \text{prices.length} \le 5000$
- $0 \le \text{prices}[i] \le 1000$

---

## 2. 动态规划

**思路：**

用**三态**区分「不持股」的两种来源，避免冷冻期后立刻买入：

- **$f[i][0]$**：第 $i$ 天结束时**不持股**，且**不是因为当天卖出**而不持股（即昨天就不持股或昨天是冷冻期），此时的最大现金。
- **$f[i][1]$**：第 $i$ 天结束时**持股**，此时的最大现金。
- **$f[i][2]$**：第 $i$ 天结束时**不持股**，且**是因为当天卖出**而不持股（即进入冷冻期），此时的最大现金。

**转移：**

- $f[i][0] = \max(f[i-1][0], f[i-1][2])$：昨天不持股（状态 0）或昨天刚卖（状态 2），今天不买。
- $f[i][1] = \max(f[i-1][1], f[i-1][0] - \text{prices}[i-1])$：昨天就持股，或昨天是状态 0（可买）今天买入。注意不能从状态 2 直接买（冷冻期）。
- $f[i][2] = f[i-1][1] + \text{prices}[i-1]$：昨天持股，今天卖出。

**边界：** 第 1 天（下标 1）：$f[1][0]=f[1][2]=0$，$f[1][1]=-\text{prices}[0]$。

**答案：** $\max_i \max(f[i][0], f[i][2])$（最后不持股的最大利润）。

**复杂度分析：**

- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$（可压成 $O(1)$，只存前一天三个状态）

**递归 DP（思路：买/不买，买则后续两天唯一；易 TLE）**

- 时间复杂度：$O(3^n)$
- 空间复杂度：$O(n)$

```C++
class Solution {
public:
    int ans;

    void dp(vector<int>& prices, int u, int sum, int val)
    {
        int n = prices.size();
        if (u >= n) return ;
        // 不买当前股票
        dp(prices, u + 1, sum, val);
        // 购买当前股票
        if (u <= n - 2 && val == -1) dp(prices, u + 1, sum, prices[u]);
        // 卖出当前股票
        if (val != -1 && val < prices[u])
        {
            sum += prices[u] - val;
            ans = max(ans, sum);
            dp(prices, u + 2, sum, -1);
        }
    }
    
    int maxProfit(vector<int>& prices) {
        dp(prices, 0, 0, -1);
        return ans;
    }
};
```

**迭代 DP（三态状态机，推荐）**

```C++
class Solution {
public:
    int f[5010][3];

    int maxProfit(vector<int>& prices) {
        int n = prices.size();
        if (n < 2) return 0;

        f[1][0] = f[1][2] = 0, f[1][1] = -prices[0];

        for (int i = 2; i <= n; i ++)
        {
            f[i][0] = max(f[i - 1][0], f[i - 1][2]);
            f[i][1] = max(f[i - 1][1], f[i - 1][0] - prices[i - 1]);
            f[i][2] = f[i - 1][1] + prices[i - 1];
        }

        int ans = 0;
        for (int i = 1; i <= n; i ++)
            ans = max(ans, max(f[i][0], f[i][2]));
        return ans;
    }
};
```
