# [1200. 最小绝对差](https://leetcode.cn/problems/minimum-absolute-difference/description/)

> **知识点**：排序、遍历

## 1. 排序+遍历

本题要求找出数组中**最小绝对差**对应的所有数对。常规做法是先排序，然后依次检查相邻元素。

- 首先将 $arr$ 进行排序，这样相邻元素的差即为所有数对中最小的可能差值。
- 遍历排序后的数组，计算所有相邻元素 $arr[i+1] - arr[i]$ 的差，并记录最小差值 $min\_diff$。
- 再次遍历，把所有差等于 $min\_diff$ 的相邻数对添加到结果中。
- 输出所有最小差值的数组对。

### 复杂度分析

- 时间复杂度: $O(n \log n)$，主要耗在排序。
- 空间复杂度: $O(n)$，用于存储结果。

### 例子

输入：`arr = [4,2,1,3]`
- 排序后 $[1,2,3,4]$
- 相邻差依次为：$1, 1, 1$
- 最小差$=1$，所有数对为 $[[1,2],[2,3],[3,4]]$

**Python3**

```python
class Solution:
    def minimumAbsDifference(self, arr: List[int]) -> List[List[int]]:
        arr.sort()
        ans = []
        for a, b in pairwise(arr):
            if not ans or b - a == ans[-1][1] - ans[-1][0]:
                ans.append([a, b])
            elif b - a < ans[-1][1] - ans[-1][0]:
                ans = [[a, b]]
        return ans
```
