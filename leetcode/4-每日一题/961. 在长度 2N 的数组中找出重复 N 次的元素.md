# [961. 在长度 2N 的数组中找出重复 N 次的元素](https://leetcode.cn/problems/n-repeated-element-in-size-2n-array/description/)

> **日期**：2026-01-02  
> **所用时间**：2min  
> **知识点**：数组、哈希表

## 1. 哈希表

可以利用集合（哈希表）辅助判断：

- 遍历数组，把每个元素加入集合；
- 如果集合中已经有这个数了（说明之前见过），直接返回，这个数即为结果。

由于题目已保证只有一个数重复 $N$ 次，其余只出现一次，所以第一个出现重复的即答案。

复杂度分析：

- 时间复杂度：$O(n)$，只需遍历一遍数组。
- 空间复杂度：$O(n)$，集合最多存下 $N+1$ 个不同元素。

**Python3**

```python
class Solution:
    def repeatedNTimes(self, nums: List[int]) -> int:
        s = set()
        for x in nums:
            if x in s:
                return x
            s.add(x)
```
