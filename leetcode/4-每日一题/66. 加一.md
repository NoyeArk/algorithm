# [66. 加一](https://leetcode.cn/problems/plus-one/description/)

> **日期**：2026-01-01  
> **所用时间**：4min  
> **知识点**: 模拟

## 1. 模拟

这是经典的模拟题。我们可以从末尾（最低位）开始加一，注意进位即可：

1. 从数组最后一位开始遍历：
    - 如果当前位小于9，直接加1，然后返回数组（因为不会产生进位）。
    - 如果当前位等于9，则该位变为0，产生进位，继续处理前一位。
2. 如果所有位都是9，例如 `[9,9,9]`，加1后会变为 `[0,0,0]`，此时需要在最前面插入1，得到 `[1,0,0,0]`。

复杂度分析：

- 时间复杂度：$O(n)$，最坏情况下需要遍历整个数组。
- 空间复杂度：$O(1)$，如果不算返回结果的空间。

**Python3**

```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        for i in range(len(digits) - 1, -1, -1):
            if digits[i] < 9:
                digits[i] += 1
                return digits
            digits[i] = 0
        return [1] + digits
```
