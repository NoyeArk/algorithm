# 1582. 二进制矩阵中的特殊位置

> **日期**：2026-03-04  
> **所用时间**：10min  
> **知识点**：模拟、矩阵

## 1. 题目描述

给定一个大小为 `m x n` 的**二进制矩阵** `mat`（元素仅为 0 或 1）。

若位置 `(i, j)` 满足：**该行中只有 `mat[i][j]` 为 1，且该列中只有 `mat[i][j]` 为 1**，则称该位置为**特殊位置**。

请返回矩阵中**特殊位置**的个数。

**示例 1：**

- **输入**：mat = [[1,0,0],[0,0,1],[1,0,0]]
- **输出**：1
- **解释**：只有 (1, 2) 满足：第 1 行只有该位置为 1，第 2 列只有该位置为 1。

**示例 2：**

- **输入**：mat = [[1,0,0],[0,1,0],[0,0,1]]
- **输出**：3
- **解释**：三个对角线位置 (0,0)、(1,1)、(2,2) 均为特殊位置。

**提示：**

- `m == mat.length`
- `n == mat[i].length`
- `1 <= m, n <= 100`
- `mat[i][j]` 为 0 或 1

## 2. 模拟

枚举每一行：若该行只有一个 1（即 `sum(row) == 1`），找到该 1 的列下标 `j`，再检查该列是否也只有一个 1（即该列所有行的和是否为 1）。若行、列都仅有一个 1，则该位置为特殊位置，计数加一。

复杂度分析：

- 时间复杂度：$O(m \times n)$，遍历每行并求行和、列和。
- 空间复杂度：$O(1)$，仅常数额外空间。

**Python3**

```python
class Solution:
    def numSpecial(self, mat: List[List[int]]) -> int:
        ans = 0
        for row in mat:
            if sum(row) != 1:
                continue
            j = row.index(1)
            if sum(row[j] for row in mat) == 1:
                ans += 1
        return ans
```
