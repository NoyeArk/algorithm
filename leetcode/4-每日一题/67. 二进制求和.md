# [67. 二进制求和](https://leetcode.cn/problems/add-binary/description/)

> **日期**：2026-02-15  
> **所用时间**：12min  
> **知识点**：模拟

## 1. 题目描述

给定两个**二进制字符串** `a` 和 `b`，以二进制字符串的形式返回它们的和。

**示例 1：**

```
输入：a = "11", b = "1"
输出："100"
解释：11₂ + 1₂ = 100₂
```

**示例 2：**

```
输入：a = "1010", b = "1011"
输出："10101"
解释：1010₂ + 1011₂ = 10101₂
```

**提示：**

- `1 <= a.length, b.length <= 10^4`
- `a` 和 `b` 仅由字符 `'0'` 或 `'1'` 组成
- 字符串如果不是 `"0"`，则不包含前导零

## 2. 从低位到高位模拟 + 进位

从两个字符串的**最低位**开始，同时从右往左遍历，用变量 `t` 表示当前位的和（含进位）。每次加上 `a[i]`、`b[j]` 的数值，当前位结果为 `t % 2`，进位为 `t // 2`；当两串都走完且进位为 0 时结束。结果字符串每次把新的一位拼在左侧（或先逆序再反转）。

复杂度分析：

- **时间复杂度**：$O(\max(n, m))$，其中 $n$、$m$ 为两串长度。
- **空间复杂度**：$O(\max(n, m))$，用于存储结果字符串（不计返回值本身则为 $O(1)$ 额外空间）。

**Python3**

```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        ans = ''
        t = 0
        i, j = len(a) - 1, len(b) - 1
        while i >= 0 or j >= 0 or t:
            if i >= 0:
                t += int(a[i])
            if j >= 0:
                t += int(b[j])
            ans = str(t % 2) + ans
            t //= 2
            i -= 1
            j -= 1
        return ans
```
