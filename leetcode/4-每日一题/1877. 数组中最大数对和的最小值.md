# [1877. 数组中最大数对和的最小值](https://leetcode.cn/problems/minimize-maximum-pair-sum-in-array/description/)

> **日期**：2026-01-24
> **所用时间**：2min
> **知识点**：贪心、排序

## 1. 题目描述

一个数对 $(a,b)$ 的 数对和 等于 $a + b$ 。最大数对和 是一个数对数组中最大的 数对和 。

比方说，如果我们有数对 $(1,5)$ ，$(2,3)$ 和 $(4,4)$，最大数对和 为 $\max(1+5, 2+3, 4+4) = \max(6, 5, 8) = 8$ 。

给你一个长度为 偶数 $n$ 的数组 $nums$ ，请你将 $nums$ 中的元素分成 $n / 2$ 个数对，使得：

- $nums$ 中每个元素 恰好 在 一个 数对中，且
- 最大数对和 的值 最小 。

请你在最优数对划分的方案下，返回最小的 最大数对和 。

## 2. 贪心

本题的目标是将数组分成 $n/2$ 个数对，要求每个数只能使用一次，并最小化这些数对中最大一对的和。

**思路分析：**

这是一个经典的贪心问题。为了让最大数对和最小，需要把数组排序后，依次将最小的数和最大的数配对。即第一个数和最后一个数配对，第二个数和倒数第二个数配对，以此类推。这样能保证每一对的和尽可能平均，从而使得最大和最小。

**算法步骤：**
1. 对数组进行升序排序。
2. 使用双指针，分别指向数组的首尾，每次将 `nums[i]` 和 `nums[n-1-i]` 配对。
3. 计算所有数对的和，并记录最大值。
4. 返回最大值。

**时间复杂度**: $O(n \log n)$，主要耗时在排序。  
**空间复杂度**: $O(1)$，仅使用常数空间。

**示例：**
输入：`nums = [3,5,4,2,4,6]`  
排序后：`[2,3,4,4,5,6]`  
配对：$(2,6)$、$(3,5)$、$(4,4)$，数对和分别为 $8$、$8$、$8$，最大值为 $8$。

该贪心策略能够使最大数对和最小。

**Python3**

```python
class Solution:
    def minPairSum(self, nums: List[int]) -> int:
        nums.sort()
        ans = 0
        n = len(nums)
        for i in range(n // 2):
            ans = max(ans, nums[i] + nums[n - i - 1])
        return ans
```
