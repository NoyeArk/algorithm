# [1984. 学生分数的最小差值](https://leetcode.cn/problems/minimum-difference-between-highest-and-lowest-of-k-scores/description/)

> **日期**：2026-01-25  
> **所用时间**：2min  
> **知识点**: 排序

## 1. 题目描述

给你一个 下标从 0 开始 的整数数组 nums，其中 nums[i] 表示第 i 名学生的分数。另给你一个整数 k。

从数组中选出任意 k 名学生的分数，使这 k 个分数间 最高分 和 最低分 的 差值 达到 最小化 。

返回可能的 最小差值 。

## 2. 排序

对数组排序后，任选 k 个学生分数的最小差值实际上等于一段长度为 k 的连续区间的最大值和最小值之差。  
枚举所有长度为 k 的滑动窗口，计算每组最大值与最小值的差，取最小即可。

思路如下：

1. 对 nums 升序排序。
2. 枚举所有长度为 k 的连续子数组（下标区间 [i - k + 1, i]，其中 i 从 k-1 遍历到 n-1）。
3. 每次计算该子数组最大（nums[i]）与最小（nums[i-k+1]）的差，记录最小值。

这样保证所有取法被覆盖，且复杂度仅为 $O(n \log n)$（排序）加 $O(n)$（遍历）。

复杂度分析：

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(1)$

**Python3**

```python
class Solution:
    def minimumDifference(self, nums: List[int], k: int) -> int:
        nums.sort()
        ans = inf
        for i in range(k - 1, len(nums)):
            ans = min(ans, nums[i] - nums[i - k + 1])
        return ans
```
