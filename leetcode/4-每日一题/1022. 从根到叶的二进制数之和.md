# [1022. 从根到叶的二进制数之和](https://leetcode.cn/problems/sum-of-root-to-leaf-binary-numbers/description/)

> **日期**：2026-02-24  
> **所用时间**：8min  
> **知识点**：DFS、二叉树

## 1. 题目描述

给定一棵**二叉树**，每个节点的值为 `0` 或 `1`。每一条**从根到叶**的路径表示一个二进制数（根为最高位、叶为最低位）。求所有从根到叶路径所表示二进制数的**和**。

例如路径 `1 → 0 → 1` 表示二进制 `101`，即十进制 5。

**示例 1：**

```
输入：root = [1,0,1,0,1,0,1]
输出：22
解释：(100) + (101) + (110) + (111) = 4 + 5 + 6 + 7 = 22
```

**示例 2：**

```
输入：root = [0]
输出：0
```

**提示：**

- 树中节点数在范围 `[1, 1000]` 内
- `Node.val` 为 `0` 或 `1`

## 2. DFS 收集路径串再求和

对根做 DFS，递归时收集「从根到当前节点」的 0/1 串；到达叶节点时得到一条完整路径对应的二进制串。将所有路径串转为十进制后求和。也可在 DFS 时直接维护当前路径的数值（父节点值 × 2 + 当前节点值），到叶节点时累加。

- **时间复杂度**：$O(n)$，n 为节点数。
- **空间复杂度**：$O(n + h)$，n 为节点数，h 为树高。

**Python3**

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def sumRootToLeaf(self, root: Optional[TreeNode]) -> int:
        def dfs(root):
            if not root:
                return []
            res = dfs(root.left) + dfs(root.right)
            return [str(root.val) + s for s in res] if res else [str(root.val)]
        return sum(int(s, 2) for s in dfs(root))
```
