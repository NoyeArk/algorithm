# 1784. 检查二进制字符串字段

> **日期**：2026-03-06  
> **所用时间**：5min  
> **知识点**：模拟、字符串

## 1. 模拟

遍历字符串，统计「连续 1 的段」的个数：若出现从 `'0'` 到 `'1'` 的过渡，说明新开了一段 1，计数加一。初始时若首字符为 `'1'` 则已有一段。最终若段数小于 2 则返回 `true`。可用 `pairwise(s)` 枚举相邻字符对，当 `c2 == '1'` 且 `c1 != c2` 时计数加一。

复杂度分析：

- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$

**Python3**

```python
class Solution:
    def checkOnesSegment(self, s: str) -> bool:
        cnt = int(s[0] == '1')
        for c1, c2 in pairwise(s):
            if c2 == '1' and c1 != c2:
                cnt += 1
        return cnt < 2
```

## 2. 脑筋急转弯

至多一段连续 1 等价于：字符串中**不出现**子串 `"01"`（否则 0 后面又出现 1，即第二段 1）。因此直接判断 `"01" not in s` 即可。

复杂度分析：

- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$

**Python3**

```python
class Solution:
    def checkOnesSegment(self, s: str) -> bool:
        return "01" not in s
```
