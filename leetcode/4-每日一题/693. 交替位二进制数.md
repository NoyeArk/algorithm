# [693. 交替位二进制数](https://leetcode.cn/problems/binary-number-with-alternating-bits/description/)

> **日期**：2026-02-18  
> **所用时间**：11min  
> **知识点**：位运算 / 模拟

## 1. 题目描述

给定一个正整数 `n`，请你检查它的二进制表示是否满足：所有**相邻**的两个数字都不同。即二进制表示中不存在连续相同的 0 或 1。

**示例 1：**

```
输入：n = 5
输出：true
解释：5 的二进制表示是 "101"，相邻位 1-0、0-1 均不同，满足交替。
```

**示例 2：**

```
输入：n = 7
输出：false
解释：7 的二进制表示是 "111"，存在连续相同的 1，不满足交替。
```

**示例 3：**

```
输入：n = 11
输出：false
解释：11 的二进制表示是 "1011"，第二、三位均为 1，不满足交替。
```

**提示：**

- `1 <= n <= 2^31 - 1`

## 2. 字符串模拟

将整数转为二进制字符串，用 `pairwise` 遍历相邻字符，若存在相邻两位相同则返回 `False`，否则返回 `True`。

- 时间复杂度: $O(\log n)$（二进制位数）
- 空间复杂度: $O(\log n)$（存储二进制字符串）

**Python3**

```python
class Solution:
    def hasAlternatingBits(self, n: int) -> bool:
        for a, b in pairwise(str(bin(n))):
            if a == b:
                return False
        return True
```
