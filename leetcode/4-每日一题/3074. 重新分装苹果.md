# [3074. 重新分装苹果](https://leetcode.cn/problems/apple-redistribution-into-boxes/description/)

> **日期**：2025-12-24
> **所用时间**：2min
> **知识点**：贪心、排序

## 1. 贪心

本题核心是典型的贪心——想“最少选多少个盒子能装下所有苹果”，那肯定每次都优先选还能装进最多苹果的盒子！原因很简单：大盒子优先填才能用尽空间，避免后面小盒子容量不足导致浪费。

复杂度分析：

- 时间复杂度：$O(n \log n)$，主要为排序时间。
- 空间复杂度：$O(n)$，主要为差值数组。

**Python3**

```python
class Solution:
    def minimumBoxes(self, apple: List[int], capacity: List[int]) -> int:
        total = sum(apple)
        capacity.sort(reverse=True)

        s = 0
        for i, c in enumerate(capacity):
            s += c
            if s >= total:
                return i + 1
```
