## 1. 构造反码

先把整数 \(n\) 转成二进制字符串，然后从高位到低位依次遍历每一位：
如果当前位是 `0`，在答案里加一位 `1`；如果当前位是 `1`，在答案里加一位 `0`，相当于逐位构造补码。

复杂度分析：

- 时间复杂度：$O(\log n)$
- 空间复杂度：$O(1)$

**Python3**

```python
class Solution:
    def bitwiseComplement(self, n: int) -> int:
        ans = 0
        for c in bin(n)[2:]:
            ans = (ans << 1) + 1 - int(c)
        return ans
```
