# 1758. 生成交替二进制字符串的最少操作数

> **日期**：2026-03-05  
> **所用时间**：6min  
> **知识点**：模拟、贪心

## 1. 只有两种情况

交替字符串只有两种可能：从下标 0 起为 `0101...`（偶数位为 0、奇数位为 1）或 `1010...`（偶数位为 1、奇数位为 0）。分别统计将 `s` 变成这两种目标时需要的修改次数，取较小值即可。遍历时对下标 `i` 判断 `s[i]` 是否与目标相同，不同则计数加一。

复杂度分析：

- 时间复杂度：$O(n)$，一次遍历。
- 空间复杂度：$O(1)$。

**Python3**

```python
class Solution:
    def minOperations(self, s: str) -> int:
        ans0 = ans1 = 0

        for i, c in enumerate(s):
            if i % 2 == 0:
                ans0 += int(c != '0')
                ans1 += int(c != '1')
            else:
                ans0 += int(c != '1')
                ans1 += int(c != '0')
        return min(ans0, ans1)
```
