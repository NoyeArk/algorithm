# LCR 076. 数组中的第 K 个最大元素

> **日期**：2024-12-23、2026-03-05  
> **所用时间**：12min  
> **知识点**：排序、快速选择

## 1. 库函数排序

最简单的做法就是将整个数组排序后返回倒数第 $k$ 个元素即可。

- 时间复杂度: $O(nlogn)$
- 空间复杂度: $O(1)$

**Python3**

```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        return sorted(nums)[-k]
```

## 2. 快速选择（基于快排划分）

利用快排的 partition：每次划分后若枢轴下标等于 `n - k`，则枢轴即为第 k 大，否则只递归一侧。期望时间复杂度 $O(n)$，最坏 $O(n^2)$；空间 $O(\log n)$ 递归栈。

**Python3**

```python
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        n = len(nums)

        def parititon(l, r):
            i, j = l, r
            while i < j:
                while i < j and nums[i] <= nums[r]:
                    i += 1
                while i < j and nums[j] >= nums[r]:
                    j -= 1
                if i < j:
                    nums[i], nums[j] = nums[j], nums[i]
                else:
                    nums[i], nums[r] = nums[r], nums[i]
            return i

        def quick_sort(l, r):
            if l < r:
                mid = parititon(l, r)
                if mid == n - k:
                    return
                quick_sort(l, mid - 1)
                quick_sort(mid + 1, r)
        quick_sort(0, n - 1)
        return nums[n - k]
```
