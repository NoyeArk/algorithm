# 334. 递增的三元子序列

## 1. 预处理左小右大

预处理 `mx[i]` 表示 `nums[i..n-1]` 的最大值；从左向右维护 `mi` 表示 `nums[0..i-1]` 的最小值。枚举中间位置 `i`（$1 \le i \le n-2$），若 `mi < nums[i] < mx[i+1]` 则存在满足条件的三元组，直接返回 `true`。

复杂度分析：

- 时间复杂度：$O(n)$
- 空间复杂度：$O(n)$（存 `mx` 数组）

**Python3**

```python
class Solution:
    def increasingTriplet(self, nums: List[int]) -> bool:
        n = len(nums)
        mx = [nums[-1]] * n
        for i in range(n - 2, -1, -1):
            mx[i] = max(mx[i + 1], nums[i])
        
        mi = nums[0]
        for i in range(1, n - 1):
            if mi < nums[i] < mx[i + 1]:
                return True
            mi = min(mi, nums[i])
        return False
```

## 2. 贪心 + 二分（最长递增子序列思路）

维护一个潜在递增序列 `f`，遍历 `nums` 时用二分查找 `x` 的插入位置：若插入位置为 2（即已有两个更小元素且 `x` 比第二个大），则说明存在长度为 3 的递增子序列，返回 `true`；否则按贪心策略更新 `f`（同 300 题 patience sorting 思路，只关心长度是否达到 3）。

复杂度分析：

- 时间复杂度：$O(n \log n)$
- 空间复杂度：$O(1)$（`f` 长度至多为 3）

**Python3**

```python
class Solution:
    def increasingTriplet(self, nums: List[int]) -> bool:
        f = []
        for x in nums:
            i = bisect.bisect_left(f, x)
            if i == 2:
                return True
            if i == len(f):
                f.append(x)
            else:
                f[i] = x
        return False
```
