# [228. 汇总区间](https://leetcode.cn/problems/summary-ranges/description/)

> **日期**：2025-12-31
> **所用时间**：5min
> **知识点**: 模拟

## 1. 模拟

1. **遍历数组**，维护区间的起点 `st`。
2. 若当前数字与下一个数字不连续，则说明当前区间结束，把区间 `[st, x]` 加入答案（`st == x` 加入单个数，否则用区间表示）。
3. 最后加上尾部区间。

复杂度分析：

- 时间复杂度：$O(n)$，只需遍历一遍数组。
- 空间复杂度：$O(1)$，只需常数辅助变量。

核心模拟点就是及时发现区间断开的位置，合理处理只有一个数字的区间与多数字区间的输出。

**Python3**

```python
class Solution:
    def summaryRanges(self, nums: List[int]) -> List[str]:
        if not nums:
            return []

        ans = []
        st = nums[0]
        for x, y in pairwise(nums):
            if x + 1 == y:
                continue
            ans.append(str(st) + "->" + str(x) if st != x else str(x))
            st = y
        ans.append(str(st) + "->" + str(nums[-1]) if st != nums[-1] else str(st))
        return ans
```
