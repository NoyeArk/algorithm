## 1. 哈希表

使用 `Counter` 统计两个字符串中每个字符出现的次数，比较两个 `Counter` 是否相等。

- 时间复杂度: $O(n)$
- 空间复杂度: $O(n)$

**Python3**

```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        return Counter(s) == Counter(t)
```
