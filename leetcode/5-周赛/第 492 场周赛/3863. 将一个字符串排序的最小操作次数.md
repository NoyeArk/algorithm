# 3863. 将一个字符串排序的最小操作次数

> **日期**：2026-03-12  
> **所用时间**：3min  
> **知识点**：分类讨论

## 1. 贪心 + 分类讨论

若串已有序（任意相邻字符非严格递增）则答案为 0。若长度为 2 且无序，则无法在有限次操作内排好序，返回 -1。否则记全局最小字符为 `mi`、最大为 `mx`：若最小字符已在首位或最大字符已在末位，说明一次操作即可（把最小移到最前或把最大移到最后）；若 `mi` 或 `mx` 出现在中间某个位置，则两次操作可完成；否则需要三次操作。

复杂度分析：

- 时间复杂度：$O(n)$
- 空间复杂度：$O(1)$

**Python3**

```python
class Solution:
    def minOperations(self, s: str) -> int:
        if all(x <= y for x, y in pairwise(s)):
            return 0
        
        if len(s) == 2:
            return -1
        
        mi, mx = min(s), max(s)
        if s[0] == mi or s[-1] == mx:
            return 1
        
        if any(c in (mi, mx) for c in s[1:-1]):
            return 2
        
        return 3
```
